## 不应扩大的结论

- 构建、测试和 mock 联调不证明 DHT22、BH1750、I²C、GPIO、树莓派系统、数据库持久化、网络、JWT、桌面客户端或报警阈值已在实物稳定运行。
- 数据库 `devices.status` 是人工维护字段，不是设备心跳、在线探测或物理状态反馈。
- API 返回成功只说明服务已接受或处理了请求，不说明传感器、继电器、外部设备或网络状态真实有效。
- 本项目不属于气象仪器、环境安全系统、消防/报警系统、医疗设备、生产控制或远程运维平台。
