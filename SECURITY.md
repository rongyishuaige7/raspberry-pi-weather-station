# 安全说明

## 部署边界

- `raspi/.env`、数据库口令、JWT 密钥和管理员口令只能保存在本机或受控部署环境，绝不能提交、截图、写入 Issue 或日志。
- 本仓默认将 Flask 绑定到 `127.0.0.1`。如确有局域网联调需求，应显式设置受控 LAN 地址，并自行配置网络隔离、反向代理、TLS、访问控制和密钥轮换。
- 此项目的 HTTP API、JWT 实现、MySQL 和桌面客户端是教学原型，不构成生产级身份、安全审计、远程监控或数据保护方案。
- 传感器、阈值、模拟数据、网页/API 响应和桌面界面均不能被表述为环境安全、预警送达、设备在线、传感器准确或持续可用的证明。

## 报告方式

请勿公开 API Token、数据库连接串、SSID、密码、私网地址、用户数据或真实传感器记录。可以通过 GitHub 私密安全报告，或联系仓库维护者公开资料中提供的渠道，说明复现条件、影响范围与最小证据。
